# Design Styles Library

## Overview
When a user asks for a design deliverable without specifying a style, don't default to generic — propose 2-3 concrete directions from this library. Each philosophy has a distinctive signature the user can recognize and choose.

## When to Use
- User says "make it look good" (no specific direction)
- Brainstorming visual identity for a new project
- Comparing aesthetic options for a rebrand
- Teaching a user about design possibilities

## The 5 Schools × 4 Philosophies

### School 1: Minimal
Focus: reduction, whitespace, clarity.

**1. Swiss / International Style**
- Grid: strict 12-column
- Typography: Helvetica/sans-serif only, bold hierarchy
- Color: black + white + 1 accent
- Layout: asymmetric but grid-aligned
- Vibe: clean, authoritative, timeless

**2. Japanese Minimalism (Ma 間)**
- Grid: generous asymmetric whitespace ("Ma" = negative space IS the design)
- Typography: thin weight, large tracking
- Color: near-white + charcoal + natural accent (clay, moss)
- Layout: off-center focal points
- Vibe: calm, contemplative, premium

**3. Scandinavian Clean**
- Grid: centered with wide margins
- Typography: rounded sans-serif, medium weight
- Color: soft pastels (dusty blue, warm gray, muted green)
- Layout: card-based, even spacing
- Vibe: friendly, approachable, warm

**4. Neo-Brutalist Minimal**
- Grid: intentionally broken grid, large blocks
- Typography: monospace or chunky sans, raw weight
- Color: stark black/white + 1 loud accent (yellow, red)
- Layout: thick borders, visible structure
- Vibe: bold, honest, anti-corporate

### School 2: Editorial
Focus: magazine-quality storytelling.

**5. Classic Editorial**
- Typography: serif headlines (Playfair, Georgia) + sans body
- Color: deep navy + cream + gold accent
- Layout: multi-column with pull quotes
- Vibe: trustworthy, intellectual, established

**6. Fashion Editorial**
- Typography: ultra-thin display + contrast bold
- Color: monochrome + 1 dramatic accent
- Layout: full-bleed images, overlapping text
- Vibe: glamorous, aspirational, bold

**7. News/Data Editorial**
- Typography: condensed sans headlines, serif body
- Color: muted palette with data-accent colors
- Layout: dense information grid, sidebar + main
- Vibe: informative, credible, efficient

**8. Literary Editorial**
- Typography: elegant serif throughout, generous leading
- Color: warm paper tones + dark ink
- Layout: single-column, wide margins, drop caps
- Vibe: thoughtful, intimate, crafted

### School 3: Corporate
Focus: professionalism, trust, scalability.

**9. Enterprise Blue**
- Typography: geometric sans (IBM Plex, Inter)
- Color: navy + slate + white + blue accent
- Layout: structured sections with clear hierarchy
- Vibe: reliable, professional, scalable

**10. Startup Bold**
- Typography: heavy sans display + light body
- Color: bright primary + dark background
- Layout: hero sections, big CTAs, testimonials
- Vibe: energetic, confident, modern

**11. Consulting Elegant**
- Typography: serif display + sans body
- Color: charcoal + white + subtle gold
- Layout: presentation-style sections, data-heavy
- Vibe: prestigious, analytical, premium

**12. Government / Institution**
- Typography: accessible sans-serif, clear sizes
- Color: patriotic palette or neutral + 1 primary
- Layout: structured, predictable, ADA compliant
- Vibe: trustworthy, accessible, official

### School 4: Creative
Focus: expression, emotion, memorability.

**13. Retro / Vintage**
- Typography: slab serif or art deco display
- Color: warm earth tones, aged paper effect
- Layout: poster-style compositions
- Vibe: nostalgic, handcrafted, characterful

**14. Geometric Abstract**
- Typography: geometric sans, tight tracking
- Color: primary colors + black/white
- Layout: circle/triangle/square motifs in grid
- Vibe: artistic, playful, structured

**15. Organic / Natural**
- Typography: rounded or hand-drawn display
- Color: earth greens, warm browns, sky blue
- Layout: flowing, non-rectilinear shapes
- Vibe: sustainable, approachable, genuine

**16. Cyberpunk / Tech**
- Typography: monospace or angular display
- Color: neon on dark (cyan, magenta, lime)
- Layout: terminal-style, glitch effects, data streams
- Vibe: futuristic, edgy, technical

### School 5: Data-Driven
Focus: making numbers tell stories.

**17. Dashboard Functional**
- Typography: tabular numerals, compact sans
- Color: semantic (green=up, red=down) + neutral grid
- Layout: card grid, KPI tiles, sparklines
- Vibe: efficient, actionable, real-time

**18. Infographic Narrative**
- Typography: big headline numbers + explanatory body
- Color: thematic gradient (warm→cool for timeline)
- Layout: vertical scroll, section-by-section reveal
- Vibe: educational, shareable, story-driven

**19. Scientific / Research**
- Typography: academic serif, footnote-heavy
- Color: muted + precise data colors (colorblind-safe)
- Layout: figure-caption pairs, reference-heavy
- Vibe: rigorous, credible, peer-reviewed

**20. Presentation / Pitch**
- Typography: impact headlines, minimal body
- Color: high contrast, brand-primary dominant
- Layout: 1 idea per slide, big visuals
- Vibe: persuasive, focused, memorable

## Process

### 1. When user has no preference
Present 3 options from DIFFERENT schools:
- "Here are 3 directions: **Swiss Minimal**, **Startup Bold**, **Infographic Narrative**. Which feels right?"
- Include a 1-sentence vibe description for each

### 2. When user has a vague preference
Map their words to a school:
- "Clean/simple" → School 1 (Minimal)
- "Professional" → School 3 (Corporate)
- "Creative/fun" → School 4 (Creative)
- "Data-heavy" → School 5 (Data-Driven)

### 3. Apply the chosen style
Use the style's specific signatures (typography, color, layout, grid) when generating the deliverable. Reference back: "Using Swiss International style as discussed."

## Rationalizations

| Excuse | Reality |
|--------|---------|
| "Just make it look good" | "Good" without direction = generic. Propose options |
| "We don't have a brand" | Then you're creating one. Be intentional |
| "All these styles look the same" | Show the user examples. They'll see the difference |

## Verification
- [ ] At least 2 distinct options proposed when direction unclear
- [ ] Each option includes: typography + color + layout signature
- [ ] Chosen style consistently applied across all deliverable sections
- [ ] Style name referenced in the output for future consistency
